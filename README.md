# README

### Fun little project for connection speed measurement with Rails & Vue
***Inspired by local network measurement***

#### 0. Prerequirements
```sh
$ ruby -v
# => ruby 2.6.5p114
$ node -v
# => v10.19.0
$ yarn -v
# => 1.22.5
```

#### 1. Install packages
```sh
$ yarn && bundle
```

#### 2. Enjoy!
```sh
$ rails s --no-log-to-stdout -p 3000
```
